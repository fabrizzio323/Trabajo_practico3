Lenguaje java

El lenguaje Java fue lanzado por primera vez en 1995 por Sun Microsystems, es uno de los lenguajes mas populares del mundo ya que es un lenguaje de alto nivel, ya que utiliza el paradigma orientado a objetos, es robusto, distribuido, interpretado dentro de java todos es considerado un objeto menos los tipo de datos primitivos. Actualmente en 2024 millones de desarrolladores y empresas continúan utilizando Java para una variedad de aplicaciones, desde desarrollo web hasta aplicaciones móviles y empresariales.

¿Como funciona?

En el lenguaje de programacion de Java todo codigo fuente que es escrito se guarda en un archivo llano con extenciones (.java) para luego ser compilado por el compilador Javac Todo el resultado de esta compilacion se guarda en un archivo con extenciones (.class) que contiene codigo de tipo Bytecode, ahora para poder realizar todos los procesos mencionados anteriormente utiliza la herramienta de Java Launcher.

Componentes principales de Java

JRE(Java Runtime Enviroment)

Este componente permite que se ejecuten aplicaciones de java en ordenadores

JDK(Java Development Kit)

Contiene todo los elementos de JRE y adiciona herramientas que facilitan al desarrollador a crear aplicaciones java

Principales ventajas y desventajas de Java

Ventajas

  • java tiene una alta portabilidad
  • Es un lenguaje orientado a objetos
  • Tiene una amplia biblioteca estadar
  • Java ofrece soporte de programas concurrentes y multitareas
  • Es un lenguaje con seguridad robusta

Desventajas

  • Tiene una alta curva de aprendizaje
  • Puede consumir mas memoria a comparacion de otros lenguajes
  • Las aplicaciones pueden tener un tiempo de arranque mas largo
  • Ausencia de punteros directos
  • Problemas de escalabilidad vertical

¿Que es HTML?

HTML es un lenguaje de marcado de hipertexto, sirve principalmente para darle la estructura o el esquelo a una pagina web mediante la aplicacion de etiquetas y atributos, proporciona un conjunto de elementos que permiten organizar el contenido de una pagina web de manera semantica, es un componente fundamental en el desarrollo web, ya que proporciona la base sobre la cual se construyen las páginas web. Junto con CSS (Cascading Style Sheets) para el diseño y la presentación, y JavaScript para la interactividad y la programación del lado del cliente, HTML forma parte de las tecnologías básicas utilizadas en el desarrollo web moderno.

Algunas etiquetas de HTML

Etiqueta Utilidad
< p > Para poder escribir parrafos
< img > Para poder colocar una imagen en la web
< a > Para colocar un enlace en la web
< h1 > al < h6 > Para poner titulos o encabezados
< main > Para definir el contenido principal de una web
< nav > Para indicar una barra de navegacion
< article > Define un articulo independiente dentro de la web
< seccion > Para agrupar temas tematicamente relacionados
< aside > Para definir contenido relacionado con el principal
< footer > Para definir el pie de pagina
